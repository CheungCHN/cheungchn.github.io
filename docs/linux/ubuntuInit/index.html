Ubuntu使用初始化

开启远程ssh#

插入MicroSD后开启电源，系统自动启动，首次进入默认用户名密码为：ubuntu/ubuntu之后会让你更改密码。

默认已经启动ssh服务，需要查看ip后可以远程ssh连入,ufw防火墙默认也是关闭。

开启root用户#

在终端输入命令：sudo passwd root 开启root用户并添加密码

ubuntu@ubuntu:~$ sudo passwd root
New password:
Retype new password:
passwd: password updated successfully

在终端输入命令：su 切换到root用户

添加root用户远程登陆ssh权限，编辑更改：

vim /etc/ssh/sshd_config

找到 #PermitRootLogin prohibit-password 去掉前面 # 并修改 prohibit-password 为yes。

如下： PermitRootLogin yes 保存。

在终端输入命令：systemctl restart sshd 重启ssh服务后可以用root远程登陆ssh

更改时区#

在终端输入命令：tzselect

然后选择Asia，输入4，然后enter

然后进入国家地区选择界面，这里选择China，输入9，然后enter

最后这里选择北京时间就行了，输入1，然后enter

最后再确认一下，选择yes，输入1，然后enter

然后查看电脑时区设置，终端输入命令：timedatectl

创建时区软连接，终端输入命令：

ln -sf /usr/share/zoneinfo/Asia/Shanghai /etc/localtime

查看系统时间，终端输入命令：date -R

更改apt国内源#

备份文件，终端输入：

mv /etc/apt/sources.list /etc/apt/sources.list.bak

输入命令vim /etc/apt/sources.list新增sources.list文件内容如下:

/etc/apt/sources.list
deb https://mirrors.aliyun.com/ubuntu-ports focal main restricted
deb https://mirrors.aliyun.com/ubuntu-ports focal-updates main restricted
deb https://mirrors.aliyun.com/ubuntu-ports focal universe
deb https://mirrors.aliyun.com/ubuntu-ports focal-updates universe
deb https://mirrors.aliyun.com/ubuntu-ports focal multiverse
deb https://mirrors.aliyun.com/ubuntu-ports focal-updates multiverse
deb https://mirrors.aliyun.com/ubuntu-ports focal-backports main restricted universe multiverse
deb https://mirrors.aliyun.com/ubuntu-ports focal-security main restricted
deb https://mirrors.aliyun.com/ubuntu-ports focal-security universe
deb https://mirrors.aliyun.com/ubuntu-ports focal-security multiverse

刷新软件源信息：终端输入命令：apt update

更新软件,终端输入命令：apt upgrade -y

Ubuntu版本问题

此次修改适用于ubuntu20.04arm版。 其他版本存在后面参数不适用。

设置静态IP#

修改配置文件,终端输入命令： vim /etc/netplan/50-cloud-init.yaml

/etc/netplan/50-cloud-init.yaml
network:
ethernets:
eth0:
dhcp4: false
addresses: [172.16.119.36/24]
gateway4: 172.16.119.254
nameserver:
addresses: [223.5.5.5,223.6.6.6]
optional: true
version: 2

查看是否修改正确,终端输入命令：netplan try

如果没有Error报错，直接按enter执行成功,或者之后终端输入命令：netplan apply

连接Wifi#

查看wlan网口名称,终端输入命令：ip addr

可以看到wlan网口名称为：wlan0,以下配置修改中需要用到

修改配置文件,终端输入命令： vim /etc/netplan/50-cloud-init.yaml

添加wifi信息如下:

/etc/netplan/50-cloud-init.yaml
network:
ethernets:
eth0:
dhcp4: false
addresses: [172.16.119.36/24]
gateway4: 172.16.119.254
nameservers:
addresses: [223.5.5.5,223.6.6.6]
optional: true
wifis:
wlan0:
dhcp4: true
access-points:
"Wifi的SSID":
password: "wifi密码"
version: 2

查看是否修改正确,终端输入命令：netplan try

如果没有Error报错，直接按enter执行成功,或者之后终端输入命令：netplan apply

安装lm-sensors硬件监控#

在终端输入命令：apt install -y lm-sensors

安装完成后在终端输入命令：sensors 显示温度等信息

root@ubuntu:~# sensors
cpu_thermal-virtual-0
Adapter: Virtual device
temp1: +42.8°C
rpi_volt-isa-0000
Adapter: ISA adapter
in0: N/A
