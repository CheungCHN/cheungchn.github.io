GitLab安装

Docker安装GitLab#

创建目录来存储容器持久数据：#

本地位置容器位置使用目的
/opt/gitlab/data/var/log/gitlab用于存储应用程序数据
/opt/gitlab/logs/var/log/gitlab用于存储日志
/opt/gitlab/config/var/log/gitlab用于存储 GitLab 配置文件

终端输入命令：

mkdir -p /opt/gitlab/data
mkdir -p /opt/gitlab/logs
mkdir -p /opt/gitlab/config

安装Docker命令安装#

终端输入命令：

docker run --detach \
--hostname 172.16.151.135 \
--publish 443:443 --publish 80:80 --publish 22:22 \
--name gitlab \
--restart always \
--volume /opt/gitlab/config/config:/etc/gitlab \
--volume /opt/gitlab/logs:/var/log/gitlab \
--volume /opt/gitlab/data:/var/opt/gitlab \
gitlab/gitlab-ce:latest

Ubuntu安装GitLab#

note

对于 Ubuntu 20.04，还提供arm64

安装和配置必要的依赖项#

apt update
apt install -y curl openssh-server ca-certificates tzdata

接下来安装postfix以发送通知电子邮件。弹出安装窗口请选择：Internet Site。 如果不需要请跳过此步

apt install -y postfix
systemctl start postfix

添加GitLab包存储库并安装包#

添加GitLab包存储库

curl https://gitee.com/CodeSilo/scripsgo/raw/master/linux/gitlab.sh | bash

接下来，安装GitLab。请确保域名正确设置DNS域名(测试环境可以做host)，也可以使用本机ip做URL。

EXTERNAL_URL="http://172.16.151.135" apt install -y gitlab-ce

CentOS7安装GitLab#

安装和配置必要的依赖项#

yum install -y curl policycoreutils-python openssh-server
systemctl enable sshd
systemctl start sshd
#以下命令还将在系统防火墙中打开 HTTP、HTTPS 和 SSH 访问。
firewall-cmd --permanent --add-service=http
firewall-cmd --permanent --add-service=https
systemctl reload firewalld

接下来安装postfix以发送通知电子邮件。如果不需要请跳过此步

yum install postfix
systemctl enable postfix
systemctl start postfix

GitLab配置修改#

GitLab时区修改#

终端输入命令：vim /etc/gitlab/gitlab.rb

/etc/gitlab/gitlab.rb
gitlab_rails['time_zone'] = 'Asia/Shanghai'

发送邮件配置#

终端输入命令：vim /etc/gitlab/gitlab.rb 添加或修改以下：

/etc/gitlab/gitlab.rb
#你的邮箱SMTP设置
gitlab_rails['smtp_enable'] = true
gitlab_rails['smtp_address'] = "smtp.office365.com"
gitlab_rails['smtp_port'] = 587
gitlab_rails['smtp_user_name'] = "YouMail@address.com"
gitlab_rails['smtp_password'] = "YouMailPassword"
gitlab_rails['smtp_domain'] = "smtp.office365.com"
gitlab_rails['smtp_authentication'] = "login"
gitlab_rails['smtp_enable_starttls_auto'] = true
gitlab_rails['smtp_openssl_verify_mode'] = 'peer'
#启用email
gitlab_rails['gitlab_email_enabled'] = true
gitlab_rails['gitlab_email_from'] = 'YouMail@address.com'
gitlab_rails['gitlab_email_display_name'] = 'YouMail-gitlab'

SMTP设置可以参考：configure an external SMTP server

启动GitLab

gitlab-ctl reconfigure
gitlab-ctl start

测试邮件是否成功

#需要加载时间
gitlab-rails console
Notify.test_email('xxxx@qq.com', 'Message Subject', 'Message Body').deliver_now

常用命令#

gitlab-ctl start # 启动所有 gitlab 组件；
gitlab-ctl stop # 停止所有 gitlab 组件；
gitlab-ctl restart # 重启所有 gitlab 组件；
gitlab-ctl status # 查看服务状态；
gitlab-ctl reconfigure # 启动服务；
vim /etc/gitlab/gitlab.rb # 修改默认的配置文件；
gitlab-rake gitlab:check SANITIZE=true --trace # 检查gitlab；
gitlab-ctl tail # 查看日志；
