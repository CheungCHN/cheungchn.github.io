Supervisord安装使用

安装Supervisord#

yum install -y epel-release
yum install -y supervisor
systemctl start supervisord
systemctl enable supervisord

配置需要管理的程序软件#

例如prometheus的配置如下： vim /etc/supervisord.d/demo.ini

/etc/supervisord.d/demo.ini
[program:prometheus]
directory = /usr/local/prometheus
command = /usr/local/prometheus/prometheus --config.file=/usr/local/prometheus/prometheus.yml --storage.tsdb.path=/data/prometheus --web.enable-admin-api --web.enable-lifecycle --web.external-url='http://172.16.118.118:9090'
autostart = true
startretries = 10
autorestart = true
redirect_stderr = true
stdout_logfile = /var/log/prometheus/prometheus.log
stdout_logfile_maxbytes = 50MB
stdout_logfile_buckups = 20

创建配置中日志的路径#

日志路径必须存在 mkdir -p /var/log/prometheus

启动软件#

supervisorctl
>update
>start prometheus
>exit
