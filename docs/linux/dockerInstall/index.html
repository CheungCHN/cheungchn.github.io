Docker安装

Ubuntu安装#

卸载旧版本#

终端输入命令 apt remove docker docker-engine docker.io containerd runc

设置存储库#

1、更新包索引和安装包以允许通过 HTTPS 使用存储库

apt update
apt install -y \
apt-transport-https \
ca-certificates \
curl \
gnupg-agent \
software-properties-common

2、添加 Docker 的官方 GPG 密钥：

curl -fsSL https://mirrors.aliyun.com/docker-ce/linux/ubuntu/gpg | sudo apt-key add -

3、使用以下命令设置稳定存储库。若要添加夜间或测试存储库，请将单词或（或两者）添加到下面的命令中的单词之后。nightly、test、stable

  • x86_64/amd64
  • armhf
  • arm64
add-apt-repository \
"deb [arch=amd64] https://mirrors.aliyun.com/docker-ce/linux/ubuntu \
$(lsb_release -cs) \
stable"

安装Docker#

终端输入命令：

apt update
apt install -y docker-ce docker-ce-cli containerd.io

验证安装#

终端输入命令：docker version

配置Hub镜像加速#

终端输入命令： vim /etc/docker/daemon.json

编辑或添加以下字段：

/etc/docker/daemon.json
/*
注释：可根据自己的位置选择以下链接：
https://registry.cn-shanghai.aliyuncs.com
https://registry.cn-hangzhou.aliyuncs.com
*/
{
"registry-mirrors": ["https://registry.cn-shanghai.aliyuncs.com"]
}

重启Docker#

终端输入命令：systemctl restart docker

Centos安装#

卸载旧版本#

终端输入命令： yum remove docker docker-common docker-selinux docker-engine

安装依赖#

终端输入命令： yum install -y yum-utils device-mapper-persistent-data lvm2

下载Repo文件#

终端输入命令： wget -O /etc/yum.repos.d/docker-ce.repo https://mirrors.aliyun.com/docker-ce/linux/centos/docker-ce.repo

安装Docker#

终端输入命令：

yum makecache fast
yum install -y docker-ce docker-ce-cli containerd.io

验证安装#

终端输入命令：docker version

配置Hub镜像加速#

终端输入命令： vim /etc/docker/daemon.json

编辑或添加以下字段：

/etc/docker/daemon.json
/*
注释：可根据自己的位置选择以下链接：
https://registry.cn-shanghai.aliyuncs.com
https://registry.cn-hangzhou.aliyuncs.com
*/
{
"registry-mirrors": ["https://registry.cn-shanghai.aliyuncs.com"]
}

重启Docker#

终端输入命令：systemctl restart docker

caution

本次安装使用皆在 root 下执行
