树莓派/RaspberryPi系统安装

硬件设备需求#

启动必须配件

  • 树莓派4B主板
  • 5V/3A电源
  • MicroSD储存卡

可选配件

  • Micro转HDMI转接线
  • 键盘
  • 散热外壳、风扇
  • 读卡器
  • U盘类外接存储

利用RaspberryPiImage烧录系统到SD卡#

下载并安装RaspberryPiImage

Download for Windows

Download for macOS

Download for Ubuntu for x86

插上MicroSD卡到你的电脑并打开RaspberryPiImage

点击 CHOOSE OS 选择你要安装的系统 例如：推荐选择Ubuntu x64 Server20.04,高于此版本可能会出现docker安装失败

点击 CHOOSE SD Card 选择你要安装到的SD卡

点击 WRITE 开始安装系统到SD卡

USB启动

2020-09-03 之前版本的硬件，出厂eeprom默认只从tf卡启动。因此，我们大部分人拿到的rpi 4b.都是只能从tf卡启动的。需要从tf卡启动后升级eeprom再打开usb启动。eeprom 设置成 BOOT_ORDER=0xf41 即可自动usb启动。 官方文档参考链接
