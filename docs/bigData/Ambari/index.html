Ambari

简介#

Version : 2.7.5.0 Apache Ambari 是用于预配、管理和监视 Apache Hadoop 群集的工具。Ambari 由一组 RESTful API 和基于浏览器的管理界面组成。

Ambari 使系统管理员能够：

  • 预配 Hadoop 群集 Ambari 提供了一个易于使用的分步向导，用于跨任何数量的主机安装 Hadoop 服务。 Ambari 处理群集的 Hadoop 服务的配置。

  • 管理 Hadoop 群集 Ambari 提供用于在整个群集中启动、停止和重新配置 Hadoop 服务的中央管理。

  • 监视 Hadoop 群集 Ambari 提供用于监视 Hadoop 群集的运行状况和状态的仪表板。 当需要您注意时，Ambari 将发送电子邮件（例如，节点关闭、剩余磁盘空间不足等）。

系统要求#

依赖安装#

yum install y unzip wget gcc* python-devel* openssl

最大打开文件要求#

建议的最大打开文件描述符数为 10000 或更多。要检查当前值集以查看打开文件描述符的最大数量，请在每个主机上执行以下 shell 命令：

ulimit -Sn

ulimit -Hn

如果输出不大于 10000，请运行以下命令以使其设置为合适的默认值：

ulimit -n 10000

设置hostname并绑定hosts#

hostnamectl set-hostname nodea

/etc/hosts
xxx.xxx.xxx.xxx nodea
xxx.xxx.xxx.xxx nodeb

设置ssh免密访问#

ssh-keygen
ssh-copy-id root@xxx.xxx.xxx.xxx

主机启用NTP#

yum install -y ntp
systemctl enable ntpd

编辑网络配置文件#

vim /etc/sysconfig/network

/etc/sysconfig/network
NETWORKING=yes
HOSTNAME=<YOURHOSTNAME>

关闭防火墙#

systemctl disable firewalld
service firewalld stop

关闭SeLinux#

setenforce 0
sed -i "s/SELINUX=enforcing/SELINUX=disabled/g" /etc/selinux/config

修改UMASK设置用户创建文件的默认权限#

echo umask 0022 >> /etc/profile

检查你当前的UMASK： umask

安装Ambari#

下载Repo#

wget http://public-repo-1.hortonworks.com/ambari/centos7/2.x/updates/2.7.4.0/ambari.repo -O /etc/yum.repos.d/ambari.repo
wget http://public-repo-1.hortonworks.com/HDP/centos7/3.x/updates/3.1.4.0/hdp.repo -O /etc/yum.repos.d/hdp.repo
wget http://public-repo-1.hortonworks.com/HDP-GPL/centos7/3.x/updates/3.1.4.0/hdp.gpl.repo -O /etc/yum.repos.d/hdp.gpl.repo

安装#

yum install -y ambari-server

设置引导#

ambari-server setup

OK to continue [y/n] (y)? Customize user account for ambari-server daemon [y/n] (n) Enter choice (1): 1 Enable Ambari Server to download and install GPL Licensed LZO packages [y/n] (n)? y Enter advanced database configuration [y/n] (n)?

安装、配置、部署集群#

启动服务器#

  • 启动服务 ambari-server start
  • 检查进程 ambari-server status
  • 停止服务 ambari-server stop

访问网站#

http://yourIP:8080 账号：admin/admin

集群安装向导#

选择：LAUNCH INSTALL WIZARD

命名群集#

输入群集名字（不要使用空格或特殊字符，并限制12字符内）

选择版本#

现下载3.1.4VDF：http://public-repo-1.hortonworks.com/HDP/centos7/3.x/updates/3.1.4.0/HDP-3.1.4.0-315.xml

闲着下拉 add version 上传刚下载的VDF文件 选择public repositories next

Install the Ambari Agent. yum install ambari-agent Using a text editor, configure the Ambari Agent by editing the ambari-agent.ini file. as shown in the following: vi /etc/ambari-agent/conf/ambari-agent.ini [server] hostname=your.ambari.server.hostname url_port=8440 secured_url_port=8441 Start the agent on every host in your cluster. ambari-agent start The agent registers with the Server on start.
